EVALUATION METHODOLOGY
----------------------------------------------------------------------------------------------------
This study employs an objective performance frequency analysis for ranking:

Ranking Approach:
  The ranking system quantifies the frequency with which each configuration achieves
  optimal performance (rank 1) across multiple evaluation metrics. This approach
  eliminates the need for subjective weight assignment while providing transparent,
  reproducible results suitable for academic publication.

Performance Scoring:
  • Rank 1 (Optimal Performance):     3 points per occurrence
  • Rank 2 (Second-Best Performance): 2 points per occurrence
  • Rank 3 (Third-Best Performance):  1 point per occurrence

Primary Ranking Criteria (in hierarchical order):
  1. Frequency of Optimal Performance (number of metrics achieving rank 1)
  2. Weighted Performance Score (aggregate points across all metrics)
  3. Superior Performance Rate (percentage of metrics achieving top-3 ranking)

Rationale:
  This methodology is preferred for academic research because:
  • Objectivity: No subjective weight assignment required
  • Transparency: Clear, interpretable ranking criteria
  • Reproducibility: Fully deterministic results
  • Defensibility: Easy to justify in peer review
